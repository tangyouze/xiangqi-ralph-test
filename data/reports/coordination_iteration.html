棋子协作评估

def evaluate_coordination(board, my_color):
    bonus = 0.0

    # 双车协作 - 同行或同列
    if len(rooks) >= 2:
        r1, r2 = rooks[0], rooks[1]
        if r1.position.row == r2.position.row or \
           r1.position.col == r2.position.col:
            bonus += 30

    # 炮马配合 - 距离近时更强
    for cannon in cannons:
        for horse in horses:
            dist = manhattan_distance(cannon, horse)
            if dist <= 3:
                bonus += 15

    # 保护链 - 高价值棋子被保护
    for piece in high_value_pieces:
        defenders = count_defenders(board, piece.position)
        if defenders >= 1:
            bonus += 10
        if defenders >= 2:
            bonus += 5

    return bonus

协作策略细节

  • 双车协作： 同行或同列的双车可以相互支援
  • 炮马配合： 炮和马在近距离时能形成有效攻击
  • 保护链： 高价值棋子（车/炮/马）有保护更安全
  • 协作权重： 协作加分只占总评估的 15%，避免过度依赖
