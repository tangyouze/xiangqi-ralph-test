# 象棋残局库

用于 AI 训练和测试的标准象棋残局集合。

## 使用说明

- 所有残局红方先走
- 红方 10 步内必胜
- FEN 格式：`<棋盘> -:- r r`
- 标准象棋（无暗子）

## 棋子符号

| 字母 | 红方 | 黑方 |
|------|------|------|
| K/k | 帅 | 将 |
| R/r | 车 | 车 |
| H/h | 马 | 马 |
| C/c | 炮 | 炮 |
| E/e | 相 | 象 |
| A/a | 仕 | 士 |
| P/p | 兵 | 卒 |

## 备注

1. FEN 格式：`<棋盘> -:- r r`
2. 棋盘从 row 9（黑方底线）到 row 0（红方底线）
3. 列从 a 到 i（左到右）
4. 用于标准象棋测试（无暗子）
