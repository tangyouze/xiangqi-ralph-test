添加的位置评估

# 位置评估（过河、中心控制）
for piece in my_pieces:
    if not piece.is_hidden:
        # 过河加分
        if not piece.position.is_on_own_side(color):
            score += 15
        # 中心控制
        if 3 <= piece.position.col <= 5:
            score += 8

Transposition Table 结构

  • EXACT： 精确值，可直接使用
  • LOWERBOUND： 下界值（beta cutoff）
  • UPPERBOUND： 上界值（alpha cutoff）
